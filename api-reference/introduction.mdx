<Accordion title="How to make requests?">
  Requests are made to `https://fragapi.ru/v1/method-name`

  You must include the authorization header with your API key:

    `Authorization`: `Bearer XXXYourApiKeyXXX`
</Accordion>


## API Usage Examples

<CodeGroup>

```python fragapi.py
import requests

BASE_URL = "https://fragapi.ru/v1"
token = "<Your API key>"

# Make a request to the API passing your API key
r = requests.get(
  url=f"{BASE_URL}/users/me",
  headers={
    "Authorization": f"Bearer {token}",
    "Content-Type": "application/json"
  }
)
me = r.json()

# Should print data about balance, etc.
print(me)
```

```javascript fragapi.js
const baseUrl = "https://fragapi.ru/v1"
const token = "<Your API key>"

// Make a request to the API passing your API key
const response = await fetch(`${baseUrl}/users/me`, {
  method: "GET",
  headers: {
    Authorization: `Bearer ${token}`,
    "Content-Type": "application/json"
  }
})

const me = await response.json()

// Should print data about balance, etc.
console.log(me)
```
</CodeGroup>
